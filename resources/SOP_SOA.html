Standard Operating Procedure (SOP)

SOA Reconciliation Module

Apeiron Bridge simplifies your Statement of Account (SOA) reconciliation. Follow this guide to match your SOA against multiple Reference files.

Phase 1: Launch & File Selection

1. Launch the Module
On the Welcome Screen, locate the "SOA Reconciliation" card and click the green Launch button.
2. Select Base File (SOA)
Click "Browse Base File" and select your primary Statement of Account (Excel/CSV).
New: The system will automatically detect and select the Date and Amount columns for you.
3. Select Reference Files
Click "Add Reference File(s)" and select files to check against the SOA (e.g., Ref1, Ref2).
Select the Match Key for each file (or let it auto-detect).
4. Run Immediately (Recommended)
Click the green "▶ Run Reconciliation" button to process immediately.
This skips the mapping screen and uses all columns automatically!
5. Optional: Advanced Settings
Click "Advanced Settings >" only if you need fuzzy matching or specific column filtering.

Phase 2: Advanced Mapping (Optional)

Only used if you clicked "Advanced Settings".

1. Configure Base File (Left Panel)

  • Base File Columns: Review selections (Date/Amount are read-only here).

2. Configure Reference Files (Center Panel)

  • Select Reference File: Use dropdown to pick file.
  • Match Column: Verify the matching column.
  • Return Columns: Uncheck columns you don't want in the report.

3. Rules & Templates (Right Panel)

  • Fuzzy Match: Enable if IDs are messy (e.g., INV-100 vs 100).
  • Templates: Save your bespoke configuration for future use.

Phase 3: Execution & Review

1. Run Reconciliation
Click the Run Reconciliation button (bottom right).
2. View Results
Check the tabs: Detailed View (Full Data) and Discrepancy Report (Variances).
3. Export
Click "Export to Excel" to save to Downloads.
